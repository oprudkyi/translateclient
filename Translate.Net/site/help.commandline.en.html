Command Line Usage

Next few command line parameters supported :

  • -nohide - don't hide tool to tray when started. Overwrite startup setting from "Options" dialog
  • -skipsplash - allow to skip "Splash Screen" when tool starting
  • -skipchangelog - skip loading of changelog at start (useful only at development time)
  • -config PATH_TO_OPTIONS_FILE - run tool with use options file by specific path, instead of use of "c:\Documents and Settings\{USER_NAME}\Application Data\SAU KP\Translate.Net\TranslateOptions.xml"
  • -portable - enable special "portable" mode. When started with this switch tool disable non-portable features which write to registry or local folders. Disabled at least "Launch at system startup" option and automatic downloading of updates.
