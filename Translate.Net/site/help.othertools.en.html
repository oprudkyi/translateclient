Use Translate.Net from other tools



You can call Translate.Net with help of system-wide hotkeys (the other methods, like activating by mouse still not implemented).

The use of hotkeys is simple - select text you want to translate and then press shortcut (Ctrl+C+C or Ctrl+Ins+Ins, based on settings you use) then tool will be activated and selected text will be inserted into input box. Also, if you enabled automatic translation by hotkey then translation process will be started.

You can change setting of hotkeys by use "Options" dialog or via context menu (activated by right mouse click on tray icon).




You can enable\disable specific hotkey and\or enable automatic translation which will start after pressing hotkey. Please note what Translate.Net will use current translation direction for automatic translation.
